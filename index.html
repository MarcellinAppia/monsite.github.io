Qu'est ce que Github?

GITHUB est une plateforme collaborative, un site web qui vous permet d'héberger un code, un projet et le gérer. Pour se faire, GITHUB utilise les principes de contrôle de version et Git.

Le contrôle de version est le principe par lequel un développeur peut suivre et gérer les modifications apportées à son projet.
